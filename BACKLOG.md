# Backlog

## Synth Features

- **Invert mode** — flip brightness so dark=loud, bright=silent. Negative space becomes the sound.
- **Frequency detune/drift** — randomize each bin's frequency slightly off-grid. Less sterile, more organic/unsettling.
- **Edge detection mode** — Sobel/Canny filter on image before synthesis. Only outlines show in spectrogram — clean silhouettes, fewer active bins.
